## Build

To build the project, run:

```sh
swift build
```

## Format

To format Swift files, run:

```sh
swift format --in-place --recursive Sources
```

## Lint

To lint Swift files, run:

```sh
swift format lint --recursive Sources
```

Configure rules in `.swiftlint.yml` if needed.

## Run

To run the CLI:

```sh
swift run
```
